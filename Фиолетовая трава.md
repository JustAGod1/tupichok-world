# Фиолетовая трава

[[Фиолетовая трава]] — вид травы, который служит желательным кормом для [[Коровы|коров]].

По текущему канону она воспринимается коровами как более предпочтительная пища, чем [[Красная трава]].

См. также индекс [[Биология и питание]].
